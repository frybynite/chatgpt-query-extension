AI Custom Prompts transforms how you interact with AI assistants. Select any text on any webpage, press a keyboard shortcut or use the context menu, and instantly send it to ChatGPT, Gemini, a custom GPT, or a Gemini Gem with pre-configured prompts. No more copying, switching tabs, pasting, and retyping the same instructions.

Perfect for job seekers analyzing postings, researchers summarizing articles, developers reviewing code, or anyone using AI assistants regularly. Create unlimited actions with personalized prompts, assign shortcuts to each, and process text through multiple prompts simultaneously with "Run All Actions."

Everything is stored locally with full import/export support for backups and sharing. No data collection, no tracking, no external servers - just a fast, privacy-focused productivity tool.

💡 HOW IT WORKS

1. Configure your AI assistant URL:
  * ChatGPT: https://chatgpt.com/ or a custom GPT URL
  * Gemini: https://gemini.google.com/app or a Gem URL (gemini.google.com/gem/<id>)
2. Create actions with custom prompts
  * In browser enter chrome://extensions/
  * Find AI Custom Prompts
  * Click Details → Extension Options
  * Fill out the options form, including your AI URL and desired actions
  * Save your configuration
3. Navigate to a webpage and select any text
  * NOTE: refresh any open tabs after saving settings changes
4. Right-click or use keyboard shortcut to execute that action
5. A tab opens with your prompt and selected text injected — optionally auto-submitted

## Justification for Permissions

* Why does this extension need `<all_urls>` content script permission?

The extension provides customizable keyboard shortcuts that allow users to trigger custom AI prompts from any website. For these shortcuts to work consistently across all websites (job boards, documentation sites, emails, social media, etc.), the lightweight keyboard listener must be available on all pages.

The content script:
- Only listens for keyboard shortcuts
- Does not read page content or collect any data
- Only sends a message to the background script when a configured shortcut is pressed
- Is minimal (~100 lines) and can be reviewed in the open source repository

Users expect keyboard shortcuts to work everywhere, which is why we need this broad permission. The extension does not use this permission for any data collection or tracking.

Full permissions justification available at:
https://github.com/frybynite/chatgpt-query-extension/blob/main/PERMISSIONS_JUSTIFICATION.md
